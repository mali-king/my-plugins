# Stremio Enhanced: Cinematic Suite & Plugins

[![Stremio](https://img.shields.io/badge/Stremio-Enhanced-blue.svg)](https://stremio.com)

A premium collection of high-performance plugins designed to transform the Stremio experience into a world-class cinematic platform. This suite focuses on visual fidelity, UX fluidity, and deep metadata integration.

---

## 🌟 Key Features

### 🎬 Cinematic Suite
The flagship integration combining multiple core enhancements into a unified, high-performance engine.
- **Theme Engine:** Advanced OKLCH-based color math for dynamic, accessible color schemes.
- **Home Page Transformation:** A next-level, industry-leading redesign of the Stremio dashboard.
- **Stream Browser:** Replaces the standard side-panel with a rich, immersive browsing experience.
- **Data Enrichment:** Deep TMDB integration for comprehensive metadata, cast details, and ratings.

### 🛠️ Core Enhancements
- **Enhanced Video Player:** Expanded controls, PiP support, and skip-intro capabilities.
- **Dynamic Hero Banners:** Netflix-style rotating hero sections for featured content.
- **Horizontal Navigation:** Ergonomic repositioning of the main navigation bar.
- **Context Menu Fix:** Critical UI patch ensuring menus always render at the correct depth.
- **Enhanced Covers:** Intelligent aspect-ratio correction and logo overlays for library items.

## 🚀 Installation

1. **Locate Plugins Folder:**
   Navigate to your Stremio Enhanced plugins directory (usually in \`Application Support\` or \`AppData\`).
   
2. **Copy Files:**
   Drop the desired \`.js\` files into the directory.

3. **Enable Plugins:**
   Open Stremio Enhanced and enable the plugins via the settings or initializer interface.

## 🛠️ Technical Details

- **Modern JavaScript:** Built with ES6+ standards using strict mode.
- **Performance Optimized:** Includes deduplicated helpers and hash-based routing.
- **Dynamic Styling:** Heavy use of CSS-in-JS and advanced CSS variables for theme consistency.
- **Reduced Motion Support:** Respects system-level accessibility settings for animations.
